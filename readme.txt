This plugin allows you to insert cool social icons in widget area with dark and light color scheme according to your site.

== Description ==

Cool Social Icons widget plugin is very easy to use, customizable way to display icons that link visitors to your various social profiles. With it, you can easily choose which profiles to link to, customize the color scheme of your icons all from the widget form no need settings page.

== Installation ==

1. Upload the entire cool-social-icons-widget folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your Widgets menu, simply drag the widget labeled "Cool Social Icons" into a widgetized Area.
4. Configure the widget by choosing a title, color scheme , and the URLs to your various social profiles.
5. For more see screenshot section.
